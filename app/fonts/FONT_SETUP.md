# Font Setup Instructions

## Perfectly Nineties Font

**Note:** Perfectly Nineties is a commercial font by Jen Wagner Co. For this project, you need to:

1. **For Development/Testing:**
   - Download the free personal-use version from: https://www.freefontdownload.org/en/perfectly-nineties-regular.font
   - Or download from: https://www.azfonts.net/fonts/perfectly-nineties

2. **For Production/Commercial:**
   - Purchase a license from: https://creativemarket.com/jenwagnerco/6979832-Perfectly-Nineties-Nostalgic-Serif
   - Or from: https://jenwagner.co/products/perfectly-nineties-a-nostalgic-serif/

3. **Installation:**
   - Download the `.otf` or `.ttf` file
   - Place it in `app/fonts/PerfectlyNineties-Regular.otf` (or .ttf)
   - The font is already configured in the app
